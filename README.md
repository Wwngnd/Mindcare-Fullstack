# Mindcare

**MindCare** adalah layanan backend untuk fitur autentikasi, kuesioner kesehatan mental, jurnal, olahraga, stress scan berbasis AI, dan rekomendasi buku.

---

## 🌐 Demo / Base URL

- Backend: `http://localhost:3000`
- Frontend: `http://localhost:5173`
